# بوت الدردشة الشخصي 🤖

بوت دردشة بسيط وسهل الاستخدام يسمح للمستخدمين بإرسال 5 رسائل مجانية قبل طلب الاشتراك أو متابعة القناة.

## المميزات ✨

- **واجهة مستخدم حديثة وجميلة**: تصميم استجابي يعمل على جميع الأجهزة
- **نظام الرسائل المحدودة**: 5 رسائل مجانية فقط قبل طلب الاشتراك
- **ردود آلية ذكية**: البوت يرد بشكل عشوائي من قائمة ردود مختلفة
- **واجهة عربية كاملة**: دعم كامل للغة العربية مع اتجاه RTL
- **تأثيرات بصرية**: رسوم متحركة سلسة وانتقالات جميلة
- **متوافق مع الأجهزة المحمولة**: يعمل بشكل مثالي على الهواتف والأجهزة اللوحية

## كيفية الاستخدام 🚀

1. افتح ملف `index.html` في المتصفح
2. اكتب رسالتك في حقل الإدخال
3. اضغط على زر الإرسال أو اضغط Enter
4. بعد 5 رسائل، سيظهر نموذج الاشتراك

## التخصيص 🎨

يمكنك تخصيص البوت بسهولة:

### تغيير الردود الآلية
عدّل مصفوفة `botResponses` في الكود:

```javascript
const botResponses = [
    'ردك الخاص هنا! 😊',
    'رد آخر! 💪',
    // أضف المزيد من الردود
];
```

### تغيير عدد الرسائل المجانية
غيّر قيمة `MAX_FREE_MESSAGES`:

```javascript
const MAX_FREE_MESSAGES = 10; // مثلاً 10 رسائل بدلاً من 5
```

### تحديث روابط الاشتراك والقناة
عدّل دالتي `handleSubscribe()` و `handleChannel()`:

```javascript
function handleSubscribe() {
    // ضع رابط الاشتراك الخاص بك هنا
    window.open('https://your-subscription-link.com');
}

function handleChannel() {
    // ضع رابط القناة الخاص بك هنا
    window.open('https://t.me/your-channel');
}
```

## النشر على Vercel 🌐

### الخطوات:

1. **انسخ المشروع إلى GitHub**:
   ```bash
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/your-username/chatbot-project.git
   git push -u origin master
   ```

2. **انتقل إلى [Vercel.com](https://vercel.com)**

3. **اضغط على "New Project"**

4. **اختر مستودع GitHub الخاص بك**

5. **اضغط "Deploy"** - سيتم نشر المشروع تلقائياً!
